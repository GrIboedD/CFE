# CFE - Chemical Formula Editor

3D редактор для создания и визуализации структурных формул химических соединений.

## Описание

CFE (Chemical Formula Editor) — это десктопное приложение для Windows, предназначенное для создания, редактирования и визуализации трехмерных моделей структурных формул химических соединений. Проект разработан на C# с использованием .NET Framework и OpenTK для рендеринга 3D-графики.

Приложение создано в качестве учебного проекта.

## Основные возможности

- Создание и редактирование 3D моделей молекулярных структур
- Визуализация химических соединений в реальном времени
- Интерактивное управление 3D-сценой (вращение, масштабирование, перемещение камеры)
- Сохранение и загрузка созданных структур

## Технологический стек

- **Язык программирования**: C# (97.0%)
- **Графический API**: OpenGL
- **Шейдеры**: GLSL (3.0%)
- **Платформа**: .NET Framework
- **Лицензия**: Apache-2.0

## Системные требования

- Windows 7/8/10/11
- .NET Framework 4.7.2 или выше
- Видеокарта с поддержкой OpenGL 3.3+

## Установка

1. Клонируйте репозиторий
2. Откройте решение в Visual Studio 2019/2022
3. Восстановите NuGet-пакеты
4. Соберите проект в конфигурации Release

## Использование

Запустите исполняемый файл `CFE.exe` после сборки проекта. Используйте интерфейс приложения для:
- Добавления атомов и связей
- Манипулирования 3D-моделью
- Сохранения и экспорта структур
